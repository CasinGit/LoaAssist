# LoaAssist

**LoaAssist**는 로스트아크 유저를 위한 숙제 관리 오버레이 프로그램입니다.  
숙제 진행 현황을 한눈에 파악하고, 캐릭터별로 숙제를 체크할 수 있도록 도와줍니다.

## 🧩 주요 기능

- ✅ **숙제 체크 오버레이**

    - 게임 화면 위에 반투명 오버레이 형식으로 표시됩니다.
    - 주간 숙제 진행 상황을 실시간으로 확인할 수 있습니다.

- 🧙 **캐릭터 관리**

    - 여러 캐릭터를 등록하고 자동 정렬하거나 순서를 직접 변경할 수 있습니다.
    - 캐릭터별 숙제를 선택하고 해당 진행 상태를 저장합니다.

- 🛠️ **사용자 정의 레이드 설정**

    - 기본 제공되는 레이드를 그룹 기능으로 묶어서 구성 할 수 있습니다.
    - ~~기본 제공되는 주간 레이드 외에도 사용자 정의 레이드를 추가할 수 있습니다~~ _(개발 예정)_

- 💾 **로컬 데이터 저장**
    - 프로그램 설정 및 사용자 정의 데이터는 로컬 JSON 형식으로 저장됩니다.
    - 사용자가 생성한 레이드 그룹, 캐릭터, 레이드 정보와 기본 제공 데이터는 로컬 SQLite 형식으로 저장됩니다.

## 📦 다운로드

최신 버전은 [GitHub Releases](https://github.com/CasinGit/LoaAssist/releases)에서 다운로드할 수 있습니다.

- Windows 10/11 전용 (다른 OS에서는 테스트되지 않았습니다)
- 설치 없이 실행 가능한 Portable 버전 제공

## 🔧 업데이트

LoaAssist는 내장된 업데이트 시스템을 통해 자동으로 최신 버전을 확인하고 다운로드할 수 있습니다.  
또는 `설정 > 새로운 버전 확인` 버튼을 **더블 클릭**하여 수동으로 업데이트할 수 있습니다.
